# RuoYi-Vue 后端技术栈

## 项目概述

RuoYi-Vue 是一个基于Spring Boot和Vue.js的前后端分离架构的管理系统。后端采用Spring Boot框架，前端使用Vue.js框架，提供了一套完整的权限管理系统。

## 核心技术栈

### 1. Java 开发环境
- **Java 版本**: Java 1.8+
- **开发语言**: Java
- **项目构建工具**: Maven 3.1+
- **项目结构**: 多模块Maven项目

### 2. 框架核心
- **Spring Boot**: 2.5.15 - 项目核心框架，提供自动配置和快速开发能力
- **Spring Framework**: 5.3.39 - 企业级应用开发的基础框架
- **Spring Security**: 5.7.14 - 安全认证和授权框架
- **Spring Web**: Web应用开发模块
- **Spring MVC**: Web应用模型-视图-控制器框架

### 3. 数据访问层
- **MyBatis**: ORM框架，提供SQL映射和数据库操作
- **Druid**: 1.2.23 - 阿里巴巴数据库连接池，提供高性能数据库连接管理
- **PageHelper**: 1.4.7 - 分页插件，简化分页查询操作

### 4. 数据库支持
- **MySQL**: 默认数据库（从配置文件中的`helperDialect: mysql`可以看出）
- **Redis**: 6379端口 - 缓存和会话存储

### 5. Web与API
- **Tomcat**: 9.0.108 - 内嵌Web服务器
- **Swagger3**: 3.0.0 - API文档自动生成工具
- **Springfox**: 用于集成Swagger的Spring框架

### 6. 数据处理与工具
- **Fastjson2**: 2.0.58 - 阿里巴巴JSON解析器
- **Jackson**: JSON数据绑定库
- **Apache Commons**: 提供常用工具类
  - `commons-lang3`: 语言工具类
  - `commons-io`: IO操作工具类
  - `commons-pool2`: 对象池工具
- **Apache POI**: 4.1.2 - Excel文件处理工具
- **Velocity**: 2.3 - 模板引擎，用于代码生成

### 7. 安全与认证
- **JWT**: 0.9.1 - JSON Web Token，用于无状态身份验证
- **XSS防护**: 防止跨站脚本攻击
- **防重复提交**: 防止表单重复提交
- **访问限制**: 限制访问频率和次数

### 8. 缓存与会话
- **Redis**: 分布式缓存和会话存储
- **Lettuce**: Redis客户端连接池

### 9. 日志系统
- **Logback**: 1.2.13 - 日志框架

### 10. 其他组件
- **OSHI**: 6.8.3 - 系统和硬件信息获取
- **UserAgentUtils**: 1.21 - 解析客户端操作系统和浏览器信息
- **Kaptcha**: 2.3.3 - 验证码生成工具
- **Quartz**: 定时任务调度框架
- **JJWT**: 用于JWT Token的生成和解析

## 项目模块结构

### ruoyi-admin
- 项目启动模块
- 配置文件和启动类

### ruoyi-common
- 通用工具模块
- 提供基础工具类和注解

### ruoyi-framework
- 框架核心模块
- 包含安全、拦截器、配置等核心功能

### ruoyi-system
- 系统业务模块
- 包含用户、角色、菜单等系统管理功能

### ruoyi-generator
- 代码生成模块
- 提供代码自动生成功能

### ruoyi-quartz
- 定时任务模块
- 提供任务调度功能
